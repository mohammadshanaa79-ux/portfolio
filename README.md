# لعبة XO (Tic-Tac-Toe) - نسخة احترافية

## 🎮 نظرة عامة
لعبة XO حديثة وجذابة مع ذكاء اصطناعي قوي، واجهة مستخدم عصرية، وتجربة لعب سلسة.

## ✨ المميزات

### 1️⃣ الذكاء الاصطناعي
- **خوارزمية Minimax** مع Alpha-Beta Pruning لأداء مثالي
- **3 مستويات صعوبة:**
  - 😊 سهل: حركات عشوائية
  - 😐 متوسط: مزيج من الذكاء والعشوائية
  - 😈 صعب: ذكاء اصطناعي لا يُقهر
- اختيار اللعب كـ X أو O
- كشف دقيق للفائز والتعادل

### 2️⃣ واجهة المستخدم
- **تصميم عصري** مع تدرجات لونية جذابة
- **Dark Mode 🌙 و Light Mode ☀️** مع تبديل سلس
- **رسوم متحركة** عند:
  - النقر على المربعات
  - الفوز أو التعادل
  - إعادة اللعبة
- **متجاوب تماماً** مع جميع الأجهزة

### 3️⃣ المؤثرات
- **مؤثرات صوتية** لكل حركة
- **رسوم متحركة** للفوز والتعادل
- **ألوان متناغمة** مع الثيم
- **تأثيرات Hover** تفاعلية

### 4️⃣ الوظائف
- اللعب ضد الذكاء الاصطناعي
- اللعب ضد صديق على نفس الجهاز
- لوحة نتائج مستمرة
- عداد الحركات
- حفظ النتائج في Local Storage

## 📁 الملفات

### نسخة الويب (HTML/CSS/JavaScript)
- `index.html` - الواجهة الرئيسية

## 🎯 كيفية اللعب

### نسخة الويب
1. اختر وضع اللعب (ضد الذكاء الاصطناعي أو صديق)
2. إذا اخترت الذكاء الاصطناعي:
   - اختر رمزك (X أو O)
   - اختر مستوى الصعوبة
3. انقر على المربعات للعب
4. استمتع بالمؤثرات والرسوم المتحركة!

### نسخة C++
1. اختر من القائمة الرئيسية
2. أدخل رقم الخانة (0-8) للعب
3. اتبع التعليمات على الشاشة

## 📊 خوارزمية Minimax

الذكاء الاصطناعي يستخدم خوارزمية Minimax مع Alpha-Beta Pruning:

```javascript
function minimax(board, depth, isMaximizing, alpha, beta) {
    // تقييم الحالة النهائية
    if (gameOver) return score;
    
    // البحث في جميع الحركات الممكنة
    for (each possible move) {
        score = minimax(newBoard, depth + 1, !isMaximizing, alpha, beta);
        // Alpha-Beta Pruning للتحسين
        if (beta <= alpha) break;
    }
    
    return bestScore;
}
```
