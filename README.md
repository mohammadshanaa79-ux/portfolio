Tic-Tac-Toe Game (XO) | لعبة إكس أو

هذا المستودع يحتوي على تطبيق للعبة الشهيرة إكس أو (Tic-Tac-Toe) بنسختين مختلفتين: نسخة للويب (HTML/CSS/JS) ونسخة برمجية بلغة C++.

This repository contains an implementation of the famous Tic-Tac-Toe (XO) game in two different versions: a Web version (HTML/CSS/JS) and a C++ implementation.




🚀 الميزات | Features

🌐 نسخة الويب (Web Version)

•
تصميم عصري: واجهة مستخدم جذابة ومتجاوبة مع جميع الشاشات.

•
نمط اللعب: إمكانية اللعب ضد صديق (لاعب ضد لاعب) أو ضد الذكاء الاصطناعي.

•
مستويات الصعوبة: ذكاء اصطناعي بمستويات مختلفة (سهل، متوسط، صعب).

•
مؤثرات صوتية: أصوات تفاعلية عند اللعب والفوز.

•
نظام النقاط: تتبع عدد مرات الفوز والتعادل لكل لاعب.

💻 نسخة C++ (Console Version)

•
منطق قوي: تنفيذ كامل لمنطق اللعبة باستخدام البرمجة الكائنية (OOP).

•
اللعب عبر الكونسول: واجهة نصية بسيطة وسريعة.

•
خوارزميات الذكاء الاصطناعي: يتضمن منطقاً لاتخاذ القرارات من قبل الكمبيوتر.




🛠️ التقنيات المستخدمة | Technologies Used

•
Frontend: HTML5, CSS3, JavaScript (ES6+).

•
Backend Logic: C++ (Standard Library).

•
Tools: VS Code, Git/GitHub.




🎮 طريقة التشغيل | How to Run

تشغيل نسخة الويب:

1.
قم بتحميل المستودع (Clone or Download).

2.
افتح ملف index.html في أي متصفح ويب.

تشغيل نسخة C++:

1.
تأكد من وجود مترجم C++ (مثل GCC) على جهازك.

2.
قم بتجميع الملف باستخدام الأمر: g++ game.cpp -o game

3.
قم بتشغيل البرنامج: ./game




📂 هيكلة الملفات | File Structure

•
index.html: هيكل واجهة الويب.

•
style.css: تنسيقات الواجهة والألوان.

•
game.js: منطق اللعبة والذكاء الاصطناعي لنسخة الويب.

•
game.cpp: الكود المصدري لنسخة C++.

•
settings.json: إعدادات مسبقة للمشروع.
